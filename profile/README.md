## Fixing Common Seer Setup Friction

- Seer app not loading? Confirm Docker services are running, then check that the frontend port and API port are not blocked by another local process.  
- Automations failing to run? Verify the LLM API key, database connection, worker process, and any required integration credentials before rerunning the workflow.  
- Integration actions unavailable? Recheck seer gmail integration or seer github integration configuration, permissions, and environment variables.  
- Costs or approvals not behaving as expected? Review seer cost controls, seer spend caps, and seer human oversight settings in the workflow before enabling broader use.
